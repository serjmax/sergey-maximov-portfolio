# Sergey Maximov — Portfolio

Personal portfolio website for Sergey Maximov, visual designer focused on branding, creative direction and visual systems.

## Local preview

No build step or dependencies are required.

```sh
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Structure

- `index.html` — page content and metadata
- `styles.css` — layout, responsive styles and motion
- `script.js` — reveal-on-scroll behavior
- `public/images` — project covers
- `public/fonts` — local web fonts when available
- `vercel.json` — Vercel configuration

## Deployment

Import this repository into Vercel as a static site. No framework preset or build command is needed. The output directory is the project root.
